# About
[![Build Status](https://travis-ci.org/simplaapliko/About.svg)](https://travis-ci.org/simplaapliko/About)

A simple library to show information about your application.

## Usage

**Gradle**

Add the following to your `build.gradle`:
```gradle
repositories {
    maven { url 'https://jitpack.io' }
}

dependencies {
    ...

    // add specific version
    implementation 'com.github.simplaapliko:about:x.y.z@aar'

    // or a snapshot
    // implementation 'com.github.simplaapliko:about:master-SNAPSHOT@aar'
}
```
